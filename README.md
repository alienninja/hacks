# Game Hacks & Mods

Educational reverse engineering and game modification projects.

**⚠️ For Educational Purposes Only** - These modifications are intended for single-player, offline games only. Do not use on multiplayer games or games with leaderboards.

## Projects

### [Car Crash Simulator](car-crash-simulator/)
Mobile racing game with unencrypted JSON economy data.

**Features:**
- Modify vehicle prices
- Convert premium currency to free currency  
- Increase quest and daily rewards
- Reset daily rewards/quests instantly

**Tools:**
- `modify_game_config.py` - Price and reward modifier
- `reset_dailies.sh` - Daily reset script

**Requirements:** Android device, ADB, Python 3

[Read the full write-up →](car-crash-simulator/)
